## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/map-app-ISimplexity.git
```

Go to folder

```bash
 cd map-app-ISimplexity
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
