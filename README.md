## Tech Stack

- HTML 5
- CSS 3
- Javascript
- React JS
- React-leaflet

## Features

- Data Usage According to Region

## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/map-app-ISimplexity.git
```

Go to folder

```bash
 cd map-app-ISimplexity
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
